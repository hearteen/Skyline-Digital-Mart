# Skyline v2.0 - Smart Retail & Surveillance Management ERP
An advanced, high-fidelity Wholesale Management ERP System seamlessly integrated with AI Surveillance modules and a Zero-Trust Security Framework.

---

## 🚀 Project Overview
**Skyline** is a next-generation System Architecture engine designed specifically for modern wholesale merchants and smart retail ecosystems. The platform uniquely merges core ERP workflows with real-world smart surveillance models, powered by a customized  visual theme, cyan glow micro-interactions, and robust dual-language routing filters.

## 🛡️ Core Architectural Features
* **Surveillance Integration**: Built with proactive system conceptualization for seamless real-world environment tracking and person detection modules.
* **Zero-Trust Security Engine**: Implementation of session fixation shields (`session_regenerate_id`) and multi-layered verification layers to enforce a continuous authentication pattern.
* **Smart UI/UX Layer**: High-fidelity Glassmorphism visual style guide with a responsive matrix wrapper, customized inputs (`autocomplete="off"` parameter controls), and localized bilingual engine processing (English/Tamil).
* **Role-Based Dynamic Access**: Automated backend routing network separating administrative wholesale dealers (`admin/admin_dashboard.php`) from retail shopkeepers (`home.php`).
* **Cryptographic Data Protection**: Native integration of hardware-safe password hashing (`password_verify`) and robust sanitization (`mysqli_real_escape_string`) to eliminate SQL injection threat vectors.

## 🛠️ Tech Stack & Systems Environment
* **Frontend Matrix**: HTML5, CSS3 Custom Variables, FontAwesome Icons, Inter Google Font Family.
* **Backend Processing**: PHP (Session Management & Object Control Routing).
* **Database Vault**: MySQL Relational Database Service (Structured Alphanumeric Roll Identifiers and Uniquely Indexed Owner Profiles).
